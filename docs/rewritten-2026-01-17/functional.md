# 功能文档（H5 + 管理后台 + 后端）

更新日期：2026-01-17

## 1. 概览

电动车租赁分期平台由三部分组成：

- H5 租赁端（用户）：下单、KYC、订单查询。
- PC 管理后台（运营/审核）：商品与分类管理、订单审核与履约、还款与催收、短信、黑名单、数据导出。
- 后端 API（Spring Boot）：订单、商品、短信、公证、文件上传等服务。

## 2. 角色与端

- H5 用户：手机号登录，浏览商品，创建订单，补充 KYC，查看订单。
- 管理员：账号密码登录，维护商品/分类，审核订单，发起履约动作，短信提醒与逾期分层。

## 3. H5 功能

### 3.1 登录与验证

- 手机号登录，短信验证码。
- 开发/测试环境可开启固定验证码（后端可配置）。

### 3.2 商品与分类浏览

- 商品分类为三级：品牌 → 系列 → 型号。
- H5 通过标签页切换品牌/系列，通过按钮选择具体型号。
- 支持按型号筛选商品列表。

### 3.3 商品详情与下单

- 详情页展示轮播图片、车型参数、租金信息。
- 下单参数：
  - 分期数（periods）
  - 周期天数（cycleDays）
  - 押金比例（depositRatio）
  - 电池方案（batteryOption：WITH_BATTERY / WITHOUT_BATTERY）
  - 还款方式（repaymentMethod：AUTO_DEDUCT / MANUAL_TRANSFER / OFFLINE）
- 页面展示还款计划预览与押金抵扣后的尾期金额。

### 3.4 订单列表与详情

- 订单列表按时间倒序。
- 订单详情展示：状态、时间轴、还款计划、还款记录、合同/收款占位入口。
- KYC 状态提示：
  - 待补充：可跳转补充页面。
  - 审核中/已驳回/已关闭等：限制补充。

### 3.5 KYC 实名资料

三步流程：

1) 身份证上传（正反面）。
2) 人脸自拍。
3) 个人信息：实名、身份证号、就业、收入区间、住址、紧急联系人等。

## 4. 管理后台功能

### 4.1 登录

- 管理员账号密码登录。
- 账号密码来源于后端配置（生产环境通过环境变量）。

### 4.2 分类管理

- 支持三级分类（品牌/系列/型号）。
- 可新增、编辑、删除。
- 删除前需要先清空下级分类与关联商品。

### 4.3 商品管理

- 新增/编辑/删除商品。
- 多图上传：第一张为封面，可调整排序。
- 支持电池方案价格（空车/含电池）和电池规格说明。

### 4.4 订单管理（审核 + 履约）

- 审核前：可通过/驳回/调价/关闭。
- 审核后：按状态推进（交付 → 取车 → 归还 → 结清）。
- 订单详情展示 KYC、合同、收款、还款计划、状态日志。

### 4.5 还款管理

- 查看订单还款计划。
- 手动标记某期已还款。

### 4.6 提醒与短信

- 到期提醒：今天到期、3 天后到期、全部（今天+3天）。
- 逾期分层：1-3、3-10、10-30、30+ 天。
- 支持批量发送提醒/催收短信。

### 4.7 黑名单

- 可添加手机号至黑名单，禁止下单。
- 支持移除。

### 4.8 数据导出

- 订单/还款/逾期/短信记录导出 CSV。

## 5. 订单状态与流程

订单状态：

- PENDING_REVIEW（待审核）
- ACTIVE（待交付）
- DELIVERED（已交付）
- IN_USE（租赁中）
- RETURNED（已归还）
- SETTLED（已结清）
- REJECTED（已驳回）
- CLOSED（已关闭）

状态流转：

- PENDING_REVIEW → ACTIVE / REJECTED / CLOSED
- ACTIVE → DELIVERED / CLOSED
- DELIVERED → IN_USE / CLOSED
- IN_USE → RETURNED / CLOSED
- RETURNED → SETTLED / CLOSED

调价规则：仅 PENDING_REVIEW 状态允许调价。

## 6. 关键业务规则

- 分期数：3–60（后端校验）。
- 周期天数：7–60（后端校验）。
- 押金比例：0–1（double）。
- 还款计划：押金从尾期开始抵扣。
- 电池方案：
  - WITH_BATTERY 使用 rentWithBattery（若有）。
  - WITHOUT_BATTERY 使用 rentWithoutBattery（若有）。
  - 否则使用 rentPerCycle。

## 7. 权限与安全

- H5 与 Admin 使用不同类型的 Bearer Token。
- Token 有效期 3 天，后端内存保存（重启后失效）。
- H5 订单访问需校验手机号归属。
- 黑名单手机号禁止创建订单。

## 8. 文件上传与存储

- 上传接口保存至 `APP_UPLOAD_DIR` 指定目录。
- 静态访问路径为 `/uploads/**`（Nginx 反向代理）。
- 后端文件大小限制：`spring.servlet.multipart.max-file-size=5MB`。

## 9. 短信与验证码

- 支持短信验证码登录 + 还款提醒 + 逾期催收 + 手动短信。
- 支持 Mock 模式（开发环境默认启用）。
- 发送频率限制：间隔 60 秒、每日 10 条（可配置）。

## 10. 公证/合同/收款

- 电子合同与收款为预留接口（可生成示例链接）。
- 公证对接聚证（Juzheng）：
  - 发起公证申请、获取签署链接、查询状态。
  - 回调接口用于状态更新。
