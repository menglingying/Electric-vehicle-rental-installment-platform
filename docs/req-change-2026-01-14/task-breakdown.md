# 新增需求拆分任务清单

本清单基于 `confirmed-decisions.md`，用于开发排期与执行。

## A. H5 客户端（apps/h5）

- 下单资料补充
  - 新增联系人、工作、收入、住址字段
  - 所有字段必填校验，表单一次性提交
  - 就业状态单选、收入区间单选
  - 地址：省/市/区三级联动 + 详细地址输入
- 商品分类入口
  - 首页或列表新增“大分类 → 系列 → 车型”的选择路径
  - 支持分类过滤与切换
- 订单详情展示
  - 展示最新租金与分期计划
  - 若后台调价，提示“价格已更新”（文案待确认）

## B. 后台管理端（apps/admin）

- 分类管理
  - 分类树 CRUD（大分类/系列/车型）
  - 启停用与排序
- 订单调价入口
  - 仅允许“审核前”调价
  - 支持调整租金/期数/周期并预览分期计划
  - 调价记录留痕（操作人/时间/原因）
- 合同处理提示
  - 调价后提示需“重签合同”

## C. 服务端 API（services/api）

- 订单资料字段扩展
  - 新增联系人、工作、收入、住址字段
  - 新增校验规则（必填、手机号格式）
- 分类模型与接口
  - 分类树实体与查询接口
  - 商品与分类关联
- 地址数据接口
  - 省/市/区三级数据查询接口
- 字典与枚举
  - 就业状态、收入区间字典
- 调价与分期重算
  - 仅允许审核前调价
  - 重算分期计划并记录变更
- 合同处理
  - 调价触发：旧合同作废，新合同生成并重签

## D. 数据模型与迁移

- 新增字段：订单资料补充字段
- 新增表/字段：分类树、字典项、调价记录、合同作废标记
- 地址数据：省/市/区基础数据导入与维护方式

## E. 测试与验收

- 下单必填校验与错误提示
- 分类选择路径与筛选
- 调价时点限制（审核后不可调）
- 调价后分期计划重算与合同重签流程
