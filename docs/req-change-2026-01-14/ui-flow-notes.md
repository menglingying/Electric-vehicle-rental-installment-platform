# 页面流程与交互说明（新增需求）

## H5 端

### 1) 商品分类选择流程

1. 进入商品页 → 展示“大分类”入口
2. 选择大分类 → 展示对应系列列表
3. 选择系列 → 展示具体车型列表
4. 选择车型 → 进入商品详情/下单

要求：
- 分类切换时保留当前位置
- 列表支持按分类过滤

### 2) 下单资料填写

表单字段（全部必填）：
- 身份证信息：姓名、身份证号、身份证正反面、人脸自拍照
- 联系人：姓名、手机号、关系
- 工作情况：就业状态（单选）
- 收入情况：月收入区间（单选）
- 住家地址：省/市/区联动 + 详细地址

交互要点：
- 省/市/区级联选择器
- 详细地址输入框（示例：街道 + 门牌号）
- 提交前统一校验并提示

### 3) 订单详情展示

- 展示最新租金与分期计划
- 若后台调价，提示“价格已更新，已生成新合同”

## 后台端

### 1) 分类管理

- 分类树可视化管理（大类/系列/车型）
- 支持新增/编辑/停用/排序

### 2) 订单调价

- 仅在“审核前”开放调价入口
- 调价后提示：需重签合同
- 展示调价前后对比与操作留痕
