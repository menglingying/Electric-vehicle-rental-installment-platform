### 7. 短信服务 ✅ (2026-01-14 完成)

**新建文件（后端 9 个）：**
```
services/api/src/main/java/com/evlease/installment/sms/
├── SmsProperties.java        # 配置属性类
├── SmsSender.java            # 发送器接口
├── SmsResult.java            # 发送结果
├── IhuyiSmsSender.java       # 互亿无线实现
├── MockSmsSender.java        # Mock实现
├── SmsConfig.java            # 配置类
├── SmsService.java           # 统一短信服务
├── VerificationCodeStore.java # 验证码内存缓存
└── VerificationCodeService.java # 验证码服务
```

**功能特性：**
- 验证码发送：15分钟有效，60秒发送间隔，每日10次限制
- 还款提醒短信：批量发送
- 逾期催收短信：批量发送
- H5 登录页60秒倒计时

**环境配置：**
| 环境 | 配置 |
|------|------|
| 开发环境 | mock=true，固定验证码 123456 |
| 生产环境 | mock=false，互亿无线真实发送 |

**短信API：**
- APIID：C13900499
- 服务商：互亿无线
